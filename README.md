# pro-mksport-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存放，可直接通过浏览器打开访问。

仓库不针对特定域名或网站，仅作为页面资料的集中存放与版本管理场所。

## 项目简介

该仓库的主要目的是整理和存档一系列独立的 HTML 页面。每个页面均为自包含的静态文档，不依赖外部数据库或后端服务。适合用于记录、展示或分享各类信息内容。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（如样式、脚本、图片等）
├── README.md       # 本文件
└── .gitignore      # Git 忽略规则
```

- **pages/**：所有 HTML 页面均放置在此目录下，每个文件为一个独立的页面。
- **assets/**：页面中引用的外部资源，按类型或页面归类存放。

## 页面归档说明

- 每个 HTML 页面均为完整自包含的文档，可独立运行。
- 页面之间无强依赖关系，可单独浏览或引用。
- 归档内容可能随仓库更新而增加、修改或移除。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进页面内容或结构。
- 新增页面请放置于 `pages/` 目录，并尽量保持命名清晰。
- 请勿在页面中引入外部不可控资源（如未知域名下的脚本），以确保页面长期可用。
- 本仓库仅用于静态页面归档，不提供动态服务或 API。

## 使用方式

克隆仓库到本地后，直接用浏览器打开 `pages/` 下的任意 HTML 文件即可查看对应页面内容。

```bash
git clone https://github.com/your-username/pro-mksport-notes.git
cd pro-mksport-notes
# 打开 pages/ 目录下的文件
```
